# CGM 图谱解读（AGP）— 照护师参考

## AGP 报告基本要素

- **中位线（50%）**：典型日内血糖轨迹，反映整体趋势。
- **四分位带（IQR）**：血糖波动范围，带越宽波动越大。
- **TIR / TAR / TBR**：目标范围内时间、高于目标、低于目标的时间占比；三者之和为 100%。
- **夜间与日间分段**：识别夜间低血糖、黎明现象等。

## 解读原则

1. **先问临床背景**：诊断分型、用药、近期调药、妊娠等特殊状态。
2. **再指图谱特征**：高峰/低谷时段、与进餐/运动的对应关系。
3. **行动建议需个体化**：饮食、运动、监测频率、何时就医；**不代替医生调药**，不擅自给出具体胰岛素剂量。

## 常见误区

- 仅看单次数值而忽略趋势；忽视 TBR 风险；把 GMI 与实验室 HbA1c 混为一谈而不说明滞后性。
