# CGM核心知识库

## 核心定义
CGM（持续葡萄糖监测）通过传感器监测皮下组织间液葡萄糖浓度，间接反映血糖水平。核心组成：可复用发射器+一次性探头。核心价值：血糖趋势监测，而非点对点精准数值。

## 三类监测区别
- 静脉血：诊断定罪，糖尿病确诊依据
- 指尖血：日常巡逻，单次血糖监测
- CGM：战略地图，全程趋势和波动监测，每天1440个点
- 血糖变化顺序：动脉血→毛细血管血→组织间液（5~10分钟生理性延迟）

## 核心指标
- TIR：3.9~10.0mmol/L时间占比，目标>70%
- TAR 1级：10.1~13.9目标<25%；2级：>13.9目标<5%
- TBR 1级：3.0~3.8目标<4%；2级：<3.0目标<1%
- CV变异系数：≤36%推荐<33%
- GMI糖化预测值：目标<7.0%
- TIR+TAR+TBR=100%
- TIR每升10%：视网膜病变风险降64%，微量白蛋白尿降40%

## 佩戴注意
- MRI必须摘下；CT/B超/安检可佩戴
- 前2天浸润期数据不稳定
- 校准：6h内禁止，48h后空腹时建议校准
- 药物干扰：对乙酰氨基酚>4g/天、维C>500mg/天、羟基脲、甘露醇、酒精

## A级推荐人群
所有T1DM、胰岛素强化治疗T2DM、频发低血糖T2DM

## 患者引入四步法
1. 为什么要测：观察药效、发现低血糖、指导饮食运动
2. 测不了那么多：1440次vs7次
3. 不疼：软丝设计，非钢针
4. 计划：制定佩戴计划和复诊时间

## 常见质疑应对
- "准不准"：血流顺序延迟、样本差异、平稳前提、伤口愈合期
- "不怕扎手指"：指尖血无法发现趋势、夜间血糖、无症状低血糖
- "为什么多台"：AGP决策周期（基线→干预→验证）

## AGP决策周期
- 参考基线AGP：发现问题
- 临床处方AGP：干预问题
- 效果确认AGP：验证效果
