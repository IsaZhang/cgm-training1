# 小程序部署指南

## 1. 生成代码上传密钥

1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入「管理」-「开发管理」-「开发设置」
3. 找到「小程序代码上传」，点击生成密钥
4. 下载 `private.key` 文件，保存到项目根目录

## 2. 配置 IP 白名单（可选但推荐）

在微信公众平台开启 IP 白名单，添加你的服务器 IP 地址。

## 3. 上传代码

```bash
# 上传代码（默认版本 1.0.0）
npm run upload

# 指定版本号和描述
node upload.js 1.0.1 "修复语音考核bug"
```

## 4. 提交审核

上传成功后，登录微信公众平台提交审核。

## 注意事项

- `private.key` 已添加到 .gitignore，不会被提交到 git
- 密钥拥有上传权限，请妥善保管
- 遗失后需要在微信公众平台重新生成
